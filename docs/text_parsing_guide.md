Также сущность может являться прилагательным. Тогда вопросная форма будет ТАКОЙ / ТАКАЯ. 

Тип сущности будет определять формат вопроса в карточке.

| Тип сущности        | Вопросные формы              |
|---------------------|------------------------------|
| YEAR                | В ЭТОМ ГОДУ                  |
| DATE                | В ЭТУ ДАТУ                   |
| CHARACTER           | ЭТОТ ПЕРСОНАЖ, ОН            |
| BRAND               | ЭТОТ БРЕНД                   |
| NICKNAME            | ЭТО ПРОЗВИЩЕ, ТАКОЕ ПРОЗВИЩЕ |
| IDIOM               | ЭТА ИДИОМА, ЭТО ВЫРАЖЕНИЕ    |
| COUNTRY             | ЭТА СТРАНА, В ЭТОЙ СТРАНЕ    |
| CITY                | ЭТОТ ГОРОД, В ЭТОМ ГОРОДЕ    |
| STATE               | ЭТОТ ШТАТ, В ЭТОМ ШТАТЕ      |
| PERSONALIA_MALE     | ОН                           |
| PERSONALIA_FEMALE   | ОНА                          |
| GOD                 | ОН, ЭТОТ БОГ                 |
| GODDESS             | ОНА, ЭТА БОГИНИЯ             |
| MOVIE               | В ЭТОМ ФИЛЬМЕ, ЭТОТ ФИЛЬМ    |
| PAINTING            | НА ЭТОЙ КАРТИНЕ, ЭТА КАРТИНА |
| ENGRAVING           | НА ЭТОЙ ГРАВЮРЕ, ЭТА ГРАВЮРА |


Вопрос в карточке должен быть согласован в роде, числе и падеже с текстом вопроса.
