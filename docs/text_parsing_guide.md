Также сущность может являться прилагательным. Тогда вопросная форма будет ТАКОЙ / ТАКАЯ. 

Тип сущности будет определять формат вопроса в карточке.

| Тип сущности      | Вопросные формы              |
|-------------------|------------------------------|
| NUMBER            | СТОЛЬКО                      |
| YEAR              | В ЭТОМ ГОДУ                  |
| DATE              | В ЭТУ ДАТУ                   |
| CHARACTER_MALE    | ЭТОТ ПЕРСОНАЖ, ОН            |
| CHARACTER_FEMALE  | ЭТОТ ПЕРСОНАЖ, ОНА           |
| BRAND             | ЭТОТ БРЕНД                   |
| NICKNAME          | ЭТО ПРОЗВИЩЕ, ТАКОЕ ПРОЗВИЩЕ |
| IDIOM             | ЭТА ИДИОМА, ЭТО ВЫРАЖЕНИЕ    |
| COUNTRY           | ЭТА СТРАНА, В ЭТОЙ СТРАНЕ    |
| CITY              | ЭТОТ ГОРОД, В ЭТОМ ГОРОДЕ    |
| STATE             | ЭТОТ ШТАТ, В ЭТОМ ШТАТЕ      |
| PERSONALIA_MALE   | ОН                           |
| PERSONALIA_FEMALE | ОНА                          |
| GOD               | ОН, ЭТОТ БОГ                 |
| GODDESS           | ОНА, ЭТА БОГИНИЯ             |
| MOVIE             | В ЭТОМ ФИЛЬМЕ, ЭТОТ ФИЛЬМ    |
| PAINTING          | НА ЭТОЙ КАРТИНЕ, ЭТА КАРТИНА |
| ENGRAVING         | НА ЭТОЙ ГРАВЮРЕ, ЭТА ГРАВЮРА |
| SONG              | ЭТА ПЕСНЯ, В ЭТОЙ ПЕСНЕ      |

Вопрос в карточке должен быть согласован в роде, числе и падеже с текстом вопроса.


Если тип сущности определить не удалось (например, она составная), то просто заменяем в вопросе 
на соответствующую тексту вопросную форму

| Сущность           | Вопросные формы            | Почему такая форма                                                  | 
|--------------------|----------------------------|---------------------------------------------------------------------|
| фильмы Тарантино   | ТАМ                        | Связано с типом MOVIE, но слово "фильмы" уже есть в самой сущности. |
| бренд сигарет      | ОН                         | Понятие мужского рода                                               |
| магический квадрат | ОН                         | Понятие мужского рода                                               |
| СССР и Израиль     | ЭТИ СТРАНЫ, В ЭТИХ СТРАНАХ | Пара сущностей, обе из которых имеют тип COUNTRY.                   |
