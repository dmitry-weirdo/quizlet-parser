## Правила дат (как применяли)

| Ответ | Заготовка в вопросе |
|-------|---------------------|
| только год (`1941`, `1989`, `1844`) | **В ЭТОМ ГОДУ** |
| день + месяц + год (`23 августа 1968`) | **В ЭТУ ДАТУ** |
| месяц + год (`май 1962`, `осень 1968`) | **В ЭТУ ДАТУ** |

Дату/год **не вставляем** в вопрос, если спрашиваем автора, покупателя и т.п. — для даты отдельная карточка.
